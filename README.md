# Focus Moon v2.8 High-End Edition

Футуристическое приложение для продуктивности и осознанности с глубоким погружением.

## Основные возможности
- **Атмосферное погружение**: Уникальные темы «Жидкое стекло» и встроенный Ambient-микшер.
- **Умное планирование**: Персональные планы обучения и задач, генерируемые ИИ (Gemini 1.5 Flash).
- **Созвездия прогресса**: Интерактивная карта достижений, где выполненные задачи становятся звездами.
- **Локальная безопасность**: Система резервного копирования (Backup/Restore) через JSON.
- **Синхронизация**: Поддержка Google Sheets для внешнего хранения данных.
- **Премиальный дизайн**: Тщательно проработанные интерфейсы с поддержкой тем Dark и Nordic.

## Технологический стек
- **Frontend**: React 18, Vite, Tailwind CSS, Framer Motion (анимации).
- **Backend**: Node.js / Express (проксирование запросов к ИИ).
- **AI**: Google Gemini 1.5 Flash Latest.

## Быстрый старт
1. Установите зависимости: `npm install`
2. Запустите сервер разработки: `npm run dev`
3. Приложение будет доступно на порту 3000.
